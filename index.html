Twitch Stream Monitor

A small page to display the status of various twitch streams, using the TwitchTV API. Streams are shown as online, offline or not existing, and the display can be filtered.

Wiki Viewer

A single page with search box and "random" article generator. Uses the Wikipedia API. Searching a term occurs on key-up, and will return a list of pages, which are clickable and will generate a new tab with the article.

Weather App

A single page weather app using geo-location to fetch weather info for the user's current location. Uses the open-weather API.

Biography Page

A biography page on Norman Borlaug, an American biologist and humanitarian who led initiatives worldwide that contributed to the extensive increases in agricultural production termed the Green Revolution.
